## 🎨 デザインコンセプト

### カラー
- **Primary**: オレンジ（エネルギッシュな優しさ）
- **Secondary**: 青（知性）

### フォント
- **数字**: 5xl（48px）
- **説明**: base（16px）

### レイアウト
- **グリッド**: 4カラム（デスクトップ）→ 2カラム（タブレット）→ 1カラム（モバイル）

## 🚀 デプロイ方法

### 1. 環境変数の設定
```.env
API_KEY=your_microcms_api_key
NEXT_PUBLIC_GOOGLE_TAG_MANAGER_ID=your_gtm_id
```

### 2. ビルド
```bash

yarn build
```
