# newCreator.org ウェブサイト改善実装レポート

## 実装日
2025年11月28日

## 目的
newCreator.orgのウェブサイトを「より大きく見せる」「活発に活動しているように見せる」「寄付を促進する」ために、戦略的な改善を実施しました。

## 実装内容

### 1. 実績を数字で見せるセクションの追加

トップページのHeroセクション直後に、**インパクトのある数字セクション**を新設しました。

#### 追加した数字
- **1,000+ 累計参加生徒数**
- **50+ ワークショップ・授業実施回数**
- **10+ 都道府県で活動**
- **8年 活動実績**（2017年から）
- **10+ パートナー企業**
- **22件 プレスリリース配信**
- **生成AI教育のパイオニア**（2023年から）

#### デザイン特徴
- **グラデーション背景**：オレンジ（from-orange-50）から青（to-blue-50）へのグラデーション
- **大きなフォント**：数字を5xlサイズで目立たせる
- **アイコン付き**：各指標にSVGアイコンを配置
- **ホバーエフェクト**：カードにホバー時の影効果（hover:shadow-lg）
- **レスポンシブ対応**：モバイルでも見やすいレイアウト

#### コード位置
`pages/index.tsx` 18行目〜152行目

---

### 2. 寄付セクションの大幅改善

#### 改善内容

##### 2-1. 寄付の使途を具体化
従来の「活動資金として使われます」という抽象的な表現から、**金額別の具体的な使途**を明示しました。

- **¥3,000**: 1人の子どもが1ヶ月間プログラミング教室に参加できます
- **¥10,000**: 地方での出張授業1回分の交通費になります
- **¥30,000**: 新しい教材の開発費に充てられます

##### 2-2. デザイン改善
- **カラーコーディング**：オレンジと青の交互配置で視覚的な変化
- **ボーダー付きカード**：border-2で各金額を明確に区別
- **背景色**：bg-orange-50、bg-blue-50で柔らかい印象

##### 2-3. CTAボタンの強化
- **色変更**：indigo-500 → **orange-500**（エネルギッシュな優しさを表現）
- **シャドウ追加**：shadow-lg、hover:shadow-xlで立体感
- **トランジション**：transition-allでスムーズなホバー効果

#### コード位置
`pages/index.tsx` 717行目〜770行目

## 技術的な注意事項

### 環境変数
本番環境では必ず `.env` ファイルに以下を設定してください：

```
API_KEY=your_microcms_api_key
NEXT_PUBLIC_GOOGLE_TAG_MANAGER_ID=your_gtm_id
```

### ビルドとデプロイ
```bash
# 依存関係のインストール
yarn install

# 開発サーバー起動
yarn dev

# 本番ビルド
yarn build

# 本番サーバー起動
yarn start
```

### Vercelへのデプロイ
環境変数を Vercel のダッシュボードで設定してください：
- `API_KEY`
- `NEXT_PUBLIC_GOOGLE_TAG_MANAGER_ID`
